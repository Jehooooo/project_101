**Role:** End-User Manual Documentation

**Task:** Write the **User Manual** for the system — a step-by-step guide that teaches users (staff and admins) how to use the system, written in simple Filipino or English.

**Specific Instructions:**
1. Open the system (the Lead Developer will give you the link)
2. Go through these sections and document each one with steps:
   - How to **log in** (Admin Login and Staff Login)
   - How to **submit an incident report** (Staff side)
   - How to **view and update an incident** (Admin side)

5. Insert the screenshots into the document beside the steps

**Format:** Microsoft Word (.docx) or Google Docs

**Deliverables:**
- [ ] User manual document with screenshots
- [ ] At least 2 pages per major section
